Bem-vindo!

Nos CSS nós posuímos os sleetores por ID

  • Seletor por tipo: Busca elementos por uma tag HTML específica
  • Seletor por ID(#): Busca elementos através do atributos "Id"
  • Seletor por Classe (.): Busca por elementos através do atributo "class"
  • Seletor Universal (*): Seleciona todos os elementos HTML
  • Seletor de Atributo ([atrib]): Seleciona elementos que possuem um atributo especifico em nosso documento HTML e conseguimos também buscar com um valor específico
